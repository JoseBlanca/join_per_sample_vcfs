What's in each step

Step 1 — Pileup parallel

One BAM/CRAM in, one .psp out. Streams the BAM exactly once and writes the five freebayes-sufficient scalars per allele.

RolePath
CRAM/BAM inputcram_input.rs
pileup walkerpileup/
BAQbaq/
pileup → .psppileup_to_psp.rs · psp/
CLI driverpop_var_caller pileup

Step 2 — Contamination estimates optional

Reads .psp + a sample-to-batch TSV; emits per-sample contamination fractions c_s and per-batch q_b as a TOML artefact frozen for Step 3.

RolePath
batch readerbatch_assignment.rs
mixture MLEcontamination_estimation.rs
TOML artefactcontamination_artifact.rs
CLI driverpop_var_caller estimate-contamination

Step 3 — SNP calling parallel

Joins N .psp readers, filters low-complexity positions, groups overlapping events, reconstructs per-sample likelihoods, runs EM, and writes the multi-sample VCF.

RolePath
per-sample pileup joinerper_position_merger.rs
DUST filterdust_filter.rs
variant groupervariant_grouping.rs
per-group mergerper_group_merger.rs
Bayesian genotype estimatorposterior_engine.rs
cohort VCF writervcf_writer/
CLI driverpop_var_caller cohort (planned)
